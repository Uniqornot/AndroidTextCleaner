📱 Android TextProcessor
TextProcessor — это мощный и легкий инструмент для оперативной очистки и форматирования текста прямо на вашем Android-устройстве. Приложение идеально подходит для копирайтеров, редакторов и разработчиков, которым нужно быстро привести "грязный" текст в порядок.
✨ Основные возможности
•
🔠 Умная нормализация: Автоматическая замена "кривых" кавычек, тире и многоточий на правильные типографские символы.
•
🧹 Полная очистка: Удаление невидимых символов Unicode, лишних пробелов и пустых строк в один клик.
•
📝 Работа с Markdown: Быстрое удаление спецсимволов разметки (заголовки, списки, жирный текст) для получения чистого текста.
•
📏 Живая статистика: Мгновенный подсчет количества символов, слов, строк и абзацев.
•
⚡ Быстрые действия:
◦
Преобразование текста в одну строку.
◦
Исправление разрывов абзацев.
◦
Удаление лишних пробелов внутри строк.
🛠 Технологический стек
•
Language: Kotlin
•
UI Framework: Jetpack Compose (современный декларативный интерфейс)
•
Design: Material Design 3
•
Architecture: Clean logic inside a single-activity flow with state management.
🚀 Как использовать
1.
Вставьте текст: Скопируйте любой текст в буфер обмена и нажмите кнопку "Вставить" в приложении.
2.
Выберите действие: Используйте кнопки быстрого доступа (например, "Убрать пробелы") или нажмите "Полная очистка" для комплексной обработки.
3.
Результат: Перейдите во вкладку "Обработанный текст", просмотрите результат и статистику, а затем скопируйте его обратно одной кнопкой.
📦 Установка
Вы можете скачать готовый APK-файл в разделе Releases и установить его на свой Android-смартфон.
